01 Premise

The static sister of mainsail.

mainsail is the Python reference — easy to embed, easy to extend, but it always ships an interpreter or a zipapp that needs one. Topsail covers the cases mainsail can't reach.

Fully-static Linux binaries via CGO_ENABLED=0 — runs on Alpine (musl), distroless, scratch base images. One-machine cross-compile to all six platforms in a single CI job. Microsecond startup. One artifact per platform.

Same applet roster, same flag conventions, same exit codes. Cosign-signed releases with SBOMs, keyless OIDC via GitHub Actions.

03 Quickstart

Download · install · dispatch.

From a release, a distro package, source, or go install. The dispatcher checks argv[0]'s basename — symlink any applet name to call it directly.

# From a release — no Go toolchain required
$ OS=$(uname -s | tr '[:upper:]' '[:lower:]')
$ ARCH=$(uname -m | sed 's/x86_64/amd64/;s/aarch64/arm64/')
$ curl -fL -o topsail.tar.gz \
    "https://github.com/Real-Fruit-Snacks/topsail/releases/latest/download/topsail_${OS}_${ARCH}.tar.gz"
$ tar xzf topsail.tar.gz
$ ./topsail --list

# Linux distro packages — installs binary, man pages, shell completions
$ sudo dpkg -i topsail_amd64.deb         # Debian / Ubuntu
$ sudo rpm -i topsail_amd64.rpm          # Fedora / RHEL / openSUSE
$ sudo apk add --allow-untrusted topsail_amd64.apk   # Alpine

# From source — Go 1.25+
$ git clone https://github.com/Real-Fruit-Snacks/topsail && cd topsail
$ make build
$ ./topsail --list

# Multi-call dispatch
$ topsail ls -la                       # subcommand form
$ ln -s topsail ls && ./ls -la         # symlink: argv[0] basename
$ copy topsail.exe ls.exe & ls -la       :: Windows: .exe stripped before lookup

05 Architecture

Four layers, one binary.

Entry → dispatch → registry → applet. Adding an applet is one new package under applets/<name>/ plus one blank import in internal/applets/all.go.

cmd/topsail/main.go        // process entry · cli.Run
   │
   ▼
internal/cli              // argv[0] basename match · multi-call vs wrapper
internal/applet           // Register · Get · All · sync.RWMutex
internal/ioutil           // stdio package vars (test seam)

applets/
ls/         // init() registers · Main(argv) -> int
cat/        // Name · Aliases · Help · Usage
…           // 74 packages · 84 names

Duplicate names or alias collisions panic at startup so every conflict surfaces immediately.
