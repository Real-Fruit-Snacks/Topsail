What you get

One static binary, anywhere

CGO_ENABLED=0 go build produces a single executable that runs unchanged on glibc, musl/Alpine, distroless, and scratch — no libc dance, no shared libs.

Multi-call dispatch

Symlink (or copy on Windows) topsail to any applet name and the binary picks it up via argv[0]. Unknown invocations fall through to topsail <applet> ….

Microsecond startup

No interpreter cold-start. Pipelines that spawn 1000 instances feel native; CI scripts run measurably faster than their Python equivalents.

Six-platform release

Linux × amd64/arm64, macOS × amd64/arm64, Windows × amd64/arm64. Cross-compiled from one CI job; checksums signed via cosign keyless OIDC.

Hardened by default

govulncheck on every push and weekly cron, CodeQL on top of golangci-lint v2 (errcheck, gocritic, gosec, govet, revive, staticcheck). Awk runs with system() disabled. Tar/zip refuse path traversal.

SBOM + provenance

Each release archive ships with a syft-generated SBOM and the checksum file is signed with cosign keyless OIDC. Verifiable end-to-end without managing keys.

Quick start

Pre-built binary (Linux / macOS)

curl -L -o topsail.tar.gz \
  https://github.com/Real-Fruit-Snacks/topsail/releases/latest/download/topsail_$(uname -s)_$(uname -m).tar.gz
tar xzf topsail.tar.gz
./topsail --list

Build from source

git clone https://github.com/Real-Fruit-Snacks/topsail
cd topsail
make build
./topsail --list

Multi-call dispatch

topsail ls -la                    # explicit dispatch
ln -s topsail ls && ./ls -la      # symlink dispatch (Unix)
copy topsail.exe ls.exe & ls -la  # copy dispatch (Windows)

Architecture

Four moving parts. The binary's entry point imports applet packages for side effects; each applet self-registers in init(); cli.Run resolves argv[0]'s basename against the registry and dispatches.

cmd/topsail/main.go        process entry; imports applet packages, calls cli.Run
   │
   ▼
internal/cli                argv[0] basename match; multi-call vs wrapper mode
   │
   ▼
internal/applet             registry: Register, Get, All; thread-safe; panics on dup
   │
   ▼
applets/<name>/             one package per applet; init() registers; Main is the entry

Full detail in ARCHITECTURE.md: applet contract, registry semantics, dispatch flow, per-OS shims, build identification, and the divergence list.

Verify a release

cosign verify-blob \
  --certificate checksums.txt.pem \
  --signature   checksums.txt.sig \
  --certificate-identity-regexp 'https://github.com/Real-Fruit-Snacks/topsail/.*' \
  --certificate-oidc-issuer    https://token.actions.githubusercontent.com \
  checksums.txt
sha256sum -c checksums.txt --ignore-missing
